About This Project

Employee Attrition Prediction Using Artificial Neural Network (ANN)

Project Description

Aplikasi ini dikembangkan untuk membantu memprediksi kemungkinan seorang karyawan akan mengalami Employee Attrition menggunakan metode Artificial Neural Network (ANN).

Sistem memanfaatkan data karakteristik karyawan, seperti usia, jabatan, pendapatan, kepuasan kerja, work-life balance, lembur, dan berbagai faktor lainnya untuk menghasilkan prediksi secara otomatis.

Hasil prediksi diharapkan dapat membantu perusahaan dalam mengambil keputusan yang lebih cepat untuk meningkatkan retensi karyawan dan mengurangi tingkat turnover.

Project Objective

  • Predict Employee Attrition.
  • Assist HR decision making.
  • Improve employee retention.
  • Utilize Artificial Intelligence in Human Resource Management.
  • Demonstrate ANN implementation using Flask.

Technology Stack

Programming Language Python 3
Framework Flask
Machine Learning TensorFlow / Keras
Data Processing Pandas & NumPy
Preprocessing Scikit-learn
Visualization Matplotlib
Database SQLite
Frontend HTML5, CSS3, Bootstrap 5

ANN Model Overview

  • Dataset : Employee Attrition Dataset
  • Input Features : 24 Employee Attributes
  • Target : Attrition (Yes / No)
  • Algorithm : Artificial Neural Network (ANN)
  • Hidden Layer : Dense + ReLU + Dropout
  • Output Layer : Sigmoid
  • Loss Function : Binary Crossentropy
  • Optimizer : Adam
